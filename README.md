# 🚀 AI Resume Screening Chatbot

An AI-powered web application that analyzes resumes against job descriptions and provides ATS score, skill matching, and improvement suggestions.

---

## 🌟 Features

- 📊 ATS Score Calculation  
- 🔍 Resume vs Job Description Similarity  
- 🧠 Skill Matching & Missing Skills Detection  
- 🤖 AI-based Resume Suggestions (optional)  
- 🎨 Modern UI (Dark Mode + Animations)  
- 📂 Drag & Drop Resume Upload  
- 📄 Downloadable Report  
- 🔐 Login System (Frontend Demo)

---
